1. Feature Overview

Generate printable QR code stickers for personnel. When scanned (by anyone, no login required), the QR opens a mobile-optimized public profile showing the worker's certifications, emergency contacts, allergies, and compliance status. Identical architecture to the shipped Equipment QR Code feature.

Use Case

A worker is on a job site. There's an accident. A GC, OSHA inspector, or first responder needs to immediately know: What certifications does this person hold? Who is their emergency contact? Do they have allergies? They scan the QR code on the worker's hard hat or badge and get the answer instantly — no Appello account needed.

Already Shipped (Equipment QR)

  • QR code PDF generation (react-qr-code)
  • JWT-signed public link (/api/public-link/generate-link)
  • Guest page without login (/guest/equipment/[id])
  • Auth redirect (guest → admin if logged in)
  • GuestLayout component

Dev Notes — Data Sources
User fields (firstName, lastName, position, employeeNumber): prisma.user
Phone: Resolved from DocumentAttributeValue (slug: "user-phone" or "phone")
Emergency contact + allergies: prisma.employeeData (linked via user)
Certifications: prisma.userCertification with include: { Certification, TrainingProvider }
Company name/phone: prisma.instanceSettings (from activeInstance context)
Avatar: prisma.user.avatarURL (Asset relation) — get signed URL

6. Security & Privacy Matrix

Strict data exposure controls ensure only safety-relevant information is visible on the public page.

Data Field Included in JWT? Shown on Public Page? Justification
Name, Position, Employee # Yes Yes Identity verification
Photo / Avatar Yes Yes Visual identification
Certifications Yes Yes Compliance verification
Emergency Contact Yes Yes Safety-critical info
Allergies Yes Yes Medical emergency info
Worker Phone Yes Yes Contact in field
Company Name & Phone Yes Yes Employer contact
Pay Rate / Salary No No Confidential financial data
Home Address No No Privacy — PII
Date of Birth No No Privacy — PII
SSN / SIN No No Never exposed
Timesheets / HR Docs No No Internal operational data
Security Notes
JWT Secret: PUBLIC_LINK_SECRET env var (already configured for equipment QR)
Token Expiry: exp: "never" for stickers — physical objects with controlled distribution. Admin can regenerate to invalidate old tokens.
Auth Redirect: If scanned by an authenticated Appello user, they see the full admin profile, not the limited public view.
Data Staleness: JWT embeds data at generation time. Show "Data as of [date]" on public page. Sticker can be regenerated to refresh.
