Architecture

PDF Generation Stack

The actual PDF pipeline used in production. All form PDFs are generated via HTML → PSPDFKit/Nutrient cloud API. There is no @react-pdf/renderer in the production pipeline.

PDF Pipeline — Single & Batch Flow
1
Frontend Request
Client calls /api/pdf-export with reportType: "safetyForm" and id: feedbackId
2
Server-Side HTML Render
pdf-export.ts fetches form data via GraphQL, generates HTML + CSS template server-side
3
PSPDFKit / Nutrient Cloud API
HTML + CSS payload sent to https://api.nutrient.io/build — returns rendered PDF binary
4
PDF Response
Single PDF returned to the browser — for single export, this triggers download directly
5
Batch Merge Batch Only
For batch exports: repeat steps 1–4 for each selected form. Collect all PDF blobs client-side.
6
Combined Download Batch Only
pdf-merger-js (browser-side) merges all PDF blobs into one file → triggers download of combined PDF
Dev: There is NO @react-pdf/renderer in the production pipeline. All PDF generation goes through HTML → PSPDFKit API. The pdf-merger-js library is already imported and used in ExportButton — reuse the same pattern for batch form export. No changes to pdf-export.ts are needed; it already supports reportType: "safetyForm".

Step 3 — Batch PDF Export (Client-Side)

When User Clicks "Export Selected"
1
Fetch Individual PDFs
For each selected feedback ID, call /api/pdf-export with reportType: "safetyForm" and id: feedbackId
2
Collect PDF Blobs
Each call returns a PDF blob — collect all blobs in sequence
3
Merge with pdf-merger-js
Use pdf-merger-js (already available via ExportButton) to merge all blobs
4
Trigger Download
Download the merged PDF as combined-forms.pdf
const merger = new PDFMerger();
for (const feedbackId of selectedFeedbackIds) {
    const response = await fetch(`/api/pdf-export?reportType=safetyForm&id=${feedbackId}`);
    const blob = await response.blob();
    await merger.add(blob);
}
const mergedBlob = await merger.saveAsBlob();
downloadBlob(mergedBlob, "combined-forms.pdf");

Step 4 — Export Button Integration

Use existing ExportButton component with its merges prop. It already handles loading state, error handling, download trigger, and PdfFormSubmissionSettingsPanel.
