Workflow 1

Incident Lifecycle — State Machine

Every incident follows this 7-stage workflow. Permissions, required fields, and notifications are enforced at each transition.

Draft

Reporter starts entry. Min: title, type, date/time.

incident.create
Submit →

Reported

Submitted. Safety Manager notified. Critical → SMS escalation.

auto-notification
Triage →

Under Review

Safety Manager confirms severity, sets OSHA recordability, assigns investigator.

Safety Manager
Open investigation →

Investigation Started

Investigator assigned. 5 Whys, witnesses, evidence collection.

Assigned Investigator
Actions identified →

Corrective Actions Assigned

Actions created, each with assignee + due date + priority.

Investigator creates
All actions done →

Actions Completed

System auto-transitions when all actions complete. Manager notified.

System (automatic)
Final review →

Closed

All docs verified, OSHA fields confirmed, investigation complete.

Safety Manager sign-off
